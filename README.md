# College Student Placement Analysis

## Overview

This project analyzes a college student placement dataset to identify the factors that influence placement success. The analysis includes data cleaning, exploratory data analysis (EDA), statistical summaries, and data visualizations to uncover meaningful insights.

## Objective

To understand how academic performance, communication skills, internship experience, extracurricular activities, and project work impact student placement outcomes.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Key Analysis

* Data Cleaning and Preprocessing
* Summary Statistics
* Missing Value Analysis
* Placement Distribution Analysis
* CGPA vs Placement Analysis
* Communication Skills vs Placement Analysis
* Project Experience Analysis

## Key Findings

* Higher CGPA is associated with better placement outcomes.
* Strong communication skills improve employability.
* Students with more completed projects tend to have higher placement success rates.

## Outcome

The project provides data-driven recommendations that can help students improve their placement readiness and assist institutions in enhancing placement support programs.
